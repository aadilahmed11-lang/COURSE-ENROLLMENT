# COURSE-ENROLLMENT
# 🎓 Course Registration System (Streamlit + MySQL)

A full-stack **Course Management & Registration Web App** built using **Streamlit** and **MySQL**.  
It supports both **Admin** and **Student** dashboards with real-time data handling.

---

## 🚀 Features

### 👨‍🎓 Student
- 🔍 Search available courses
- 📚 View course details
- ✅ Enroll in courses
- 🎯 Track learning progress

### 🛠 Admin
- 📊 Dashboard analytics
- ➕ Add / ❌ Delete courses
- 👨‍🎓 View all students
- 📋 Monitor enrollments
- 💰 Revenue insights

---

## 🖥 Tech Stack

- **Frontend**: Streamlit
- **Backend**: Python
- **Database**: MySQL
- **Libraries**:
  - pandas
  - mysql-connector-python
